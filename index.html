Channel Search

Check the uploads of channel(s) for geotagged videos. This page is essentially YouTube Metadata Bulk but with a map and results filtered to only the geotagged ones. Recommended to instead export data from YMB and import here if you want the data for everything not geotagged too.

Submit a comma separated list of channel url(s) and/or id(s)

Accepted formats
  • https://www.youtube.com/channel/channel_id
  • https://www.youtube.com/user/username
  • https://www.youtube.com/c/custom_url (may not work, see here)
  • https://www.youtube.com/custom_url (may not work, see here)
  • Also accepts direct ids: channel_id
